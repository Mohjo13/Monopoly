# Monopoly
This project is a modular C# implementation of Monopoly built as a systems-focused programming exercise. The primary goal was to explore clean architecture, object-oriented design, and structured turn-based game logic rather than UI or graphics.

The project emphasizes maintainable code, separation of concerns, and extensibility.

**Overview**

The game runs as a console application and follows a structured turn pipeline including:

Dice rolling

Player movement

Square resolution

Economic transactions

Jail and rule handling

The architecture is organized into clearly separated core systems (Board, Players, Managers, AI, etc.), making it easy to extend with additional rules, AI behaviors, or gameplay features.

This repository serves as a demonstration of backend game architecture and C# design principles in a turn-based simulation context.
